3. What is event delegation in the DOM?

Answer: Instead of adding an event to many small elements, we add it to their parent. The event works for all child elements inside it.

4. What is event bubbling in the DOM?

Answer: When an event happens on a child element, it moves up (bubbles) to its parent and then to the whole page.
